# Conway's Game of Life (Java)

A Java application that simulates Conway’s Game of Life using a 2D grid of cells that evolve over generations.

## Features

* Random board initialization with fixed seed
* Custom board initialization
* Advance simulation by one or multiple generations
* Automatic neighbor counting
* Text-based grid visualization

## Concepts Demonstrated

* 2D Arrays
* Nested Loops
* Simulation Logic
* Object-Oriented Programming
* Boundary Handling

## Technologies

* Java
* java.util.Random
